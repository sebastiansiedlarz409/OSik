## General info
It is simple operating system for x86 CPUs.
	
## Details
What works (or works in some part):
* Boot (Real Mode > Protected Mode > Long Mode)
* Text Mode (80*25 16 colors)
* printf etc
* Keybord
* Interrupts
* Programmable Interval Timer
* Heap
* BSOD :D

## Requirements
* GCC 64bits
* NASM
* Python 3
* Bochs

## Run
To run this project:

```
$ git clone [...]
$ python build.py
```
